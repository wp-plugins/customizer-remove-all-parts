WordPress Customizer Remove All Parts, allows you to get the Customizer crap out of your WordPress install.

== Description ==

WordPress Customizer Remove All Parts allows you to get the Customizer crap out of your WordPress install. It prevents any Customizer scripts from loading and removes all links and buttons to Customizer in the dashboard and admin toolbar.

== Installation ==

Upload WP CRAP to your site or use the dashboard to Add New plugins and activate it. That's it! No options needed. Customizer is gone.
